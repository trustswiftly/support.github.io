How to Customize Documents
To customize the Documents there are a few key settings. Documents and ID collection have the most flexibility and have options to configure what is collected. For example you can request Passports, Driver Licenses, Insurance IDs, or any other ID with a name and picture.
Step 1: Open the Documents page. From the side menu, click on the Documents menu entry.
Step 2: Configure the Documents settings. On this page you will see multiple options to set. Not all settings need to be enabled and they can be left blank. To start a custom document workflow you first need to create Verify Methods. I.e. 2 entrie a Passport and Selfie. Once the methods are enabled you should create a custom Workflow to name it and set the ordering. Drag and drop the different verify methods you need and click save.
Step 3: Assign the Custom Workflow. Once you enable Document / ID verification you can select the workflow for the documents.
