Stripe Integration
Retrieve your secret keys from Stripe and save them in Trust Swiftly so we can automatically trigger verifications for payments that are defined as risky. If you do not already have a Trust Swiftly account create one before proceeding below.
First select API Keys in your stripe account
2. Copy your Secret API Key from https://dashboard.stripe.com/apikeys
Once it is created then copy your secret key.
2. Paste the Stripe API Secret Key in your Trust Swiftly management page then Update keys and then Manage.
https://NAME.trustswiftly.com/settings/setkeys
5. In the Connection section click Connect with Stripe and login. Review your verification settings for Stripe at https://NAME.trustswiftly.com/settings/stripe
Select a template to apply automatically for payments in review. If you select Approve reviews automatically the review in Stripe will be marked complete once your customer completes the verifications on Trust Swiftly. Lastly the Email notification is possible to notify the customer about the verification requirement.
